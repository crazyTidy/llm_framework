流式输出机制
工作原理：
节点的 execute 方法返回 AsyncIterator，每次 yield 都会立即发送到前端
工作流引擎会遍历节点的每个 yield 输出
app.py 将每个输出包装成 SSE 格式发送
在节点中实现流式输出：
class MyNode(BaseNode):    async def execute(self, inputs: Dict[str, Any]) -> AsyncIterator[Dict[str, Any]]:        # 每次 yield 都会立即发送到前端        for i, chunk in enumerate(data):            yield {                'result': chunk,                'node_id': self.node_id,                'chunk_index': i            }            await asyncio.sleep(0.1)  # 可选：控制发送速度

class StreamNode(BaseNode):    async def execute(self, inputs: Dict[str, Any]) -> AsyncIterator[Dict[str, Any]]:        text = inputs.get('text', '')        for i, char in enumerate(text):            yield {'result': char, 'node_id': self.node_id, 'chunk_index': i}

要点：
每个 yield 都会立即发送，无需等待节点执行完成
支持中间节点输出，任何节点的 yield 都会发送
输出格式：{'node_id': '节点ID', 'output': {yield的内容}, 'is_final': False}
在节点中多次 yield 即可实现流式输出。
